# Security Policy

## Supported versions

Security fixes are provided for the current release of `passport-custom`.
Consumers should upgrade to the newest published version before reporting an
issue that may already have been resolved.

## Reporting a vulnerability

Please do not open a public issue for a suspected vulnerability. Instead, use
GitHub's **Report a vulnerability** option on the repository's Security page:

https://github.com/mbell8903/passport-custom/security/advisories/new

Include the affected version, reproduction steps, expected impact, and any
suggested remediation. You should receive an initial response within seven
days.
